# agents-config

Personal Claude Code / agent configuration for [@yfuks](https://github.com/yfuks).

This repo holds the configuration that shapes how AI coding agents behave across all of my projects — global rules today, with skills and more to come.

## Contents

- **[`AGENTS.md`](./AGENTS.md)** — Global rules applied in every scenario, regardless of the project. These are the baseline standards (quality over development cost, engineering excellence, best practices) that individual projects layer their own instructions on top of.

## Roadmap

- **Skills** — reusable agent skills, added next.

## Usage

These files are meant to be picked up by agent tooling (e.g. Claude Code) as global instructions. Symlink or copy them into the relevant config location, or point your tooling at this repo.
